<p align="center"><b>This is a snap for DDNS Updater</b>, <i>"A program to keep DNS A and/or AAAA records updated for multiple DNS providers"</i>.</p>


# DDNS Updater Snap

## Install

```
sudo snap install ddns-updater
```

## Usage

### Service

Edit the configuration file at `/var/snap/ddns-updater/current/config.json`
and start the service with:

```
sudo snap start ddns-updater
```
